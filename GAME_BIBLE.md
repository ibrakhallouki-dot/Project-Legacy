# PROJECT LEGACY

## Vision
A game where the world permanently learns from the player's actions.

---

## Pillars

### 1. The World Remembers
Nothing resets.
Every important action leaves a permanent mark.

### 2. Every Choice Matters
There are no meaningless interactions.

### 3. Simple Rules → Infinite Possibilities
A few rules create endless gameplay.

### 4. Discovery Before Explanation
Players learn by experimenting.

### 5. Consequences Create Stories
The game never tells stories.
The player creates them.

---

## Core Question

"What happens if the world can learn?"
